## About

This blog features articles on AI/ML topics including neural networks, deep learning, NLP, and the mathematics behind modern AI systems.

## Tech Stack

- **Framework:** [Astro](https://astro.build/) - Static site generator optimized for content-driven websites
- **Styling:** Custom CSS with a dark "Technical Futurism" theme
- **Math Rendering:** KaTeX for LaTeX mathematical equations
- **Deployment:** GitHub Pages with GitHub Actions CI/CD

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
